Oscars Award Database Manager

მართავს ოსკარის დაჯილდოების მონაცემთა ბაზას (`Oscars.sqlite3`) SQLite-ის დახმარებით. პროექტი მოიცავს ძირითადი Create, Read, Update, Delete ოპერაციებს და ვიზუალიზაციას Matplotlib ბიბლიოთეკით.

მონაცემთა ბაზა

აპლიკაცია მუშაობს SQLite ბაზაზე `OscarsAward` ცხრილით. ცხრილი უნდა შეიცავდეს შემდეგ სვეტებს:



 `Name`         - TEXT     - მსახიობის/რეჟისორის სახელი და გვარი   
 `Category`     - TEXT     - ნომინაციის კატეგორია                   
 `Film`         - TEXT     - ფილმის დასახელება                     
 `Year_Ceremony`- INTEGER  - დაჯილდოების წელი                      
 `Win_Oscar`    - TEXT     - TRUE ან FALSE                         
 `Gender`       - TEXT     - Female ან Male (საჭიროა ვიზუალიზაციისთვის) 


ძირითადი ფუნქციები
`WinOrNot(Film, Name, Category)`
ამოწმებს, მოიგო თუ არა ოსკარი კონკრეტულ ფილმში კონკრეტულ კატეგორიაში მონაწილე პირმა.

`NewInformation()`
მომხმარებლისგან იღებს ახალ მონაცემებს და ამატებს მონაცემთა ბაზაში.

`UpdateInformation()`
ამარაგებს არსებულ ჩანაწერს ახალი `Win_Oscar` მნიშვნელობით, მითითებული პირობით.

`DelInformation()`
შლის კონკრეტულ ჩანაწერს მსახიობის სახელის, ფილმის, კატეგორიის, წლისა და შედეგის მიხედვით.


ვიზუალიზაცია 

პროექტი აგენერირებს 3`pie chart`ს სხვადასხვა მონაცემზე დაყრდნობით:

დაჯილდოებული ქალები vs. დაჯილდოებული კაცები
დაჯილდოებული ქალები vs. დაუჯილდოებელი ქალები
დაჯილდოებული კაცები vs. დაუჯილდოებელი კაცები

`pie chart`ის აგებისთვის გამოვიყენე სვეტი `Gender`.
